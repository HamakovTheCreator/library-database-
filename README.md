📚 Система управления библиотекой

Веб-приложение для управления библиотекой на Spring Boot с красивым интерфейсом.

 Функционал

 Книги
- Просмотр списка всех книг
- Добавление новых книг
- Поиск книг по названию и автору
- Отображение статуса (доступна/выдана)
- Быстрая выдача книги читателю

 Читатели
- Просмотр списка всех читателей
- Добавление новых читателей
- Поиск читателей по имени
- Автоматическая дата регистрации

 Выдачи
- Выдача книг читателям
- Отслеживание активных выдач
- Возврат книг
- Просмотр просроченных книг

 Статистика
- Общее количество книг
- Количество доступных книг
- Количество читателей
- Количество выданных книг

 Технологии

 Backend
- Java 21
- Spring Boot 4.0.3
- Spring Data JPA 
- Spring Web - REST API
- MySQL - база данных
- Hibernate - ORM
- Lombok - упрощение кода

Frontend
- HTML5
- CSS3 - стили с градиентами
- JavaScript - взаимодействие с API
- Адаптивный дизайн - работает на всех устройствах

Установка и запуск

Требования
- Java 21 или выше
- MySQL 8.0 или выше
- Maven (или используйте встроенный ./mvnw)
